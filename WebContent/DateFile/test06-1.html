[ form-2 ] form으로 사용자와 연결하기

input type="time" - 사용자 지역을 기준으로 시간 (시 분 초)을 지정

input type="date"- 사용자 지역을 기준으로 날짜 (연 월 일)을 지정

input type="week" -사용자 지역을 기준으로 날짜 (연주)을 지정

input type="month" -사용자 지역을 기준으로 날짜 (연 월)을 지정

input type="datetime-local" -사용자 지역을 기준으로 날짜 와 시간 (연 월 일 시 분 초)을 지정

input type="tel" -전화번호를 입력하는 폼요소 (가상 키패드 변화 가능, 전화연결)

input type="email"- email 주소 입력하는 폼요소 (브라우저에서 email 형식을 체크, 가상 키패드 변화가능)

input type="url" -사이트주소 입력하는폼요소 (브라우저에서 url 형식을 체크, 가상 키패드 변화가능)

input type="unmber" -숫자값을 지정하는 폼요소 (증감버튼)


min="n" 속성으로 최소값 지정가능
max="n" 속성으로 최대값 지정 가능
step="n" 속성으로 증감간격 지정가능.

input type="range" -범위내의 값을 선택하는 폼요소

input type="color" -색상을 선택하는 폼요소

input type="search" -검색용 입력창 (삭제버튼)

html5 에서 추가된 속성

placeholder - 입력예시를 보여줌


required - 필수 입력요소로 지정

autofocus - 자동 포커싱된 입력요소로 지정

autocomplete="on/off" - 자동완성 기능을 껴고 끔 (on이 기본)

readonly - 수정 불가한 (읽기만 가능한) 입력요소로 지정

disabled - 사용 불가한 입력요소로 지정
