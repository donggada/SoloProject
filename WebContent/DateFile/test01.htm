html 수업 첫번째

html 이란?

*Hyper Text Markup Language의 약자.
*웹페이지를 만들어주는 프로그래밍 언어. (구조를 담당)
*웹개발의 기본 기술 중 한가지.

html(웹페이지)을 만들어주는 기본테그

html    ------------  웹페이지 시작
 head    -----------  머리말 시작
   title      /title --   페이지(브라우저탭)의 제목
 /head    ----------  머리말 종료
 body     ----------  본문시작

          방분자에게 보일 컨텐츠

 /body   -----------  본문종료
/html  -------------  웹페이지 종료

*웹페이지를 저장할떄 확장자는 .htm이나 .htm1로 지정


문단
*html문서는 명령어, tag로 구성되며 tag는 대부분 각괄호 안에 들어있다.

*대부분의 tag는 시작과 종료 한 쌍으로 이루어진다.

*원칙적으로 tag의 대소문자 구별은 없으나 소문자로 쓴다.
